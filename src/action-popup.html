Clear Links

Some permissions are not granted

The extension requires the "Access your data for all websites" permission. Without this permission, the extension cannot function. This permission is necessary to display tooltips over links, as the extension needs to detect links within the web pages. If you prefer not to enable this for all websites, you can adjust your preferences in the Extension's Settings page.

While most permissions are granted during installation, this specific permission requires explicit user approval. Clicking the "Grant Permission" button will prompt a browser popup requesting the necessary permission.
